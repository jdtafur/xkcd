# Xkdc

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.0.4.

para ejecutar este proyecto clone el repositorio en su dispositivo con el siguiente comando
"git clone https://github.com/jdtafur/xkcd

instale los paquetes necesarios para que el proyecto pueda funcionar, para esto ejecute el comando
"npm install"

una vez instalado todos los paquetes necesarios ejecute el comando
"ng serve"

para acceder a través del lavegador use la siguiente ruta:
"http://localhost:4200/"
